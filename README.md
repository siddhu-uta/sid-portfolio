# Sid Portfolio

Welcome to Sid's personal portfolio website! This project is a visually striking, responsive portfolio designed to showcase skills, experience, projects, and certifications.

## 🚀 Features
- **Modern UI**: Built with React, Tailwind CSS, and shadcn/ui.
- **Responsive Design**: Fully responsive layout that looks great on desktop and mobile.
- **Dark Mode / Sleek Aesthetics**: High-end minimalist design reflecting an elegant professional style.
- **Dynamic Content Sections**: Covers Experience, Projects, Skills, Certifications, and more.

## 🛠️ Technologies
- [React](https://reactjs.org/) & [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Lucide Icons](https://lucide.dev/)
- Routing & animations via standard ecosystem libraries

## 💻 Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/siddhu-uta/sid-portfolio.git
   cd sid-portfolio
   ```

2. **Install dependencies**:
   Make sure you have Node and npm (or bun/yarn) installed.
   ```bash
   npm install
   # or
   bun install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   # or
   bun run dev
   ```

4. **Navigate** to `http://localhost:8080` to see the site in action!

## 📦 Building for Production

To create a production build, run:
```bash
npm run build
```
This will compile the application into the `dist` folder, ready to be deployed to platforms like Vercel, Netlify, or AWS.
